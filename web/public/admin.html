Staff tool

Add a funeral home

Paste a funeral home's website. This detects its platform, verifies it publishes obituaries, and opens a pull request adding it to the scrape config for review. If the platform isn't one we support yet, it opens a tracked issue for the team to look into instead — no submission is a dead-end. Only homes the newsroom has a republication arrangement with should be added.

The token is your own GitHub fine-grained PAT with Actions: read and write on this repo. It stays in your browser and is never committed.

This triggers the add-home.yml workflow. Detection runs server-side (the fetch is cross-origin and can't run in a browser). Watch progress and review the resulting PR on GitHub.
